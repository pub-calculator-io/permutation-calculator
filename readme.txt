Quickly calculate permutations (nPr) with our free Permutation Calculator. Find the exact number of ways to arrange r items from a set of n elements.

[https://www.calculator.io/permutation-calculator/](https://www.calculator.io/permutation-calculator/)

== Usage ==

Add the Permutation Calculator shortcode to your page, post or sidebar:

`[calcio_permutation_calculator]`

Add the following code to your template where you would like the Permutation Calculator to appear:

`<?php display_calcio_permutation_calculator(); ?>`

== Installation ==

1. Upload the Permutation Calculator /calcio_permutation_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Permutation Calculator plugin through the "Plugins" menu in WordPress.
